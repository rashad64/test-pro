Twenty basic terms and their definitions

This is a list of twenty terms that I thought were intersting while reading chapter one of the manuel.
  1. Input: entering data into computer
  2. Processing: performing operations on the data
  3. Output: Presenting the results
  4. Storage: saving data, programs or output for fure use
  5. Hardware: physical parts of computer system, such as the keyboard, monitor...
  6. Software: the instruction, also called computer programs, that we used to tell the
  7. Oprating system: the main component of system software that enables computer to operate to manage its activity
  8. Apllication Software: programs that enables user to perform specific tasks o na computer, such as writting, playing games.
  9. Embedded computer: a tiny computer embedded in a product and designed to perform specific tasks or function for that product
  10. Mobile device: a very small device that has build in computing or internet capacity
  11. Computer literacy: the knowledge and understaning of basic computer fundamentals
  12. Portal Web page: A Web page designed to be designated as a browser home page; typically can be customized to display personalized content
  13. Internet filtering: Using a software program or browser option to block access to particular Web pages or types of Web pages
  14. Cookie: A small file stored on a user’s hard drive by a Web server; commonly used to identify personal preferences and settings for that user.
  15. Spyware: A software program that is installed without the user’s permission and that secretly gathers information to be sent to others.
  16. Data: Raw: unorganized facts
  17. Information: Data that has been processed into a meaningful form
  18. Boot: To start up a computer
  19. Windows desktop: The background work area displayed on the screen for computers running Microsoft Windows
  20. Application software: Programs that enable users to perform specific tasks on a computer, such as writing letters or playing games; also called apps.
  21. Smartphone: A mobile device based on a mobile phone that includes Internet capabilities and can run mobile apps
  22. Media tablet: A mobile device, usually larger than a smartphone, that is typically used to access the Internet and display multimedia content.
  23. Thin client: A personal computer designed to access a network for processing and data storage, instead of performing those tasks locally
